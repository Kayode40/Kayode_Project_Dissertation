# Appendix C: Python Code

The Python implementation was developed to support the proposed machine learning evaluation framework. The complete implementation is intended to be executed in Jupyter Notebook or Google Colab.

The code consists of the following modules:

1. Importing required Python libraries (NumPy, Pandas, Scikit-learn, Matplotlib and Seaborn).
2. Loading the benchmark intrusion detection dataset.
3. Data preprocessing and feature scaling.
4. Splitting the dataset into training and testing sets.
5. Training the selected machine learning algorithms:

   * Random Forest
   * Support Vector Machine (SVM)
   * Decision Tree
   * K-Nearest Neighbour (KNN)
6. Evaluating models using:

   * Accuracy
   * Precision
   * Recall
   * F1-score
   * ROC-AUC
7. Generating confusion matrices and comparative performance charts.

**Note:** Representative code excerpts are included in this appendix, while the full implementation is maintained separately within the accompanying GitHub repository.
