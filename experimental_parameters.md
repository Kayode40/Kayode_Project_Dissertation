# Appendix D: Experimental Parameters

Table D.1 presents the experimental configuration adopted for the comparative evaluation framework.

| Parameter                 | Value                                          |
| ------------------------- | ---------------------------------------------- |
| Programming Language      | Python 3.11                                    |
| Development Environment   | Jupyter Notebook / Google Colab                |
| Machine Learning Library  | Scikit-learn                                   |
| Dataset                   | NSL-KDD (proposed benchmark dataset)           |
| Data Split                | 80% Training / 20% Testing                     |
| Cross-Validation          | 5-Fold                                         |
| Evaluation Metrics        | Accuracy, Precision, Recall, F1-score, ROC-AUC |
| Random Forest Trees       | 100                                            |
| Decision Tree Criterion   | Gini Index                                     |
| SVM Kernel                | Radial Basis Function (RBF)                    |
| KNN Neighbours            | 5                                              |
| Performance Visualisation | Matplotlib and Seaborn                         |

These parameters define the proposed experimental setup and provide a reproducible framework for future implementation and validation of the study.
